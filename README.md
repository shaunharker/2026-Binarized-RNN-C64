# Binarized Neural Networks for Legacy Hardware

Some experiments in quantized models! Nothing too exciting, just playing around.

## project contents

folder | contents
------ | ---------
C64    | c64 runtime for vanilla rnn model
GA     | genetic algorithm approach to training vanilla rnn model
QAT    | quantized aware training approach to training vanilla rnn model
LSTM   | experiments with LSTM model (haven't cracked it yet)
literature | related papers
training_data | download training data sets
